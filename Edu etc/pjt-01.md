## Git

### git 이란?

- 분산 버전 관리 시스템(DVCS, Distributed Version Control System)
- 2005년 리눅스 커널을 위한 도구로, 리누스 토르발스가 개발
- 컴퓨터 파일의 변경사항을 추적하고 여러 사용사 간 작업을 조율
